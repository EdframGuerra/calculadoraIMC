# Exercício 01

## Calculadora de IMC

Para calcularmos o IMC de alguém, podemos utilizar a seguinte fórmula:

$$ I = \frac{P}{A^2} $$

Onde,

**P:** peso da pessoa

**A:** altura da pessoa

**I:** IMC

#### Exemplo:

Se minha altura é 1,79m e meu peso, 76kg, qual é meu IMC?
